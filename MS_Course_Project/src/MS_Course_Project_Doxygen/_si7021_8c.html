◆ si7021_i2c_open()

void si7021_i2c_open ( )

init all I2C_OPEN_STRUCT operation values and transitions into i2c_open()

Uses timer_delay to delay while Si7021 is given max time (80ms) to fully power up. Then calls a local struct to set values that will be passed on to i2c_open and used during its setup.

◆ si7021_read()

void si7021_read ( uint32_t  CB)

called during event scheduler in main, contains i2c_start function for SI7021 operations

called in main under if statement checking for event LETIMER0_UF_CB, input value is the defined CALLBACK value of 0x160 which will remove the specific scheduled event and call the described function. This is done in conjunction with the alternate shtc3_read() function that performs a similar sweet of i2c_start function calls.

◆ si7021_read_convert()

uint32_t si7021_read_convert ( )

converts read_result value from Si7021 into a usable value

checks static global uint_32 value read_result and converts whatever value exists within it into a usable humidity value that is relayed back into scheduled_si7021_read_cb(void).

Note
checking this value is possible by setting a breakpoint after the result variable is calculated to compare that value to assumed room humidity.
