void cmu_open ( void  )

first op called during app_peripheral_setup

enables different parts of the CMU clock tree we want use the ULFRCO oscillator to enable operation down into PG energy mode 3 level

// Disable the LFRCO oscillator CMU_OscillatorEnable(cmuOsc_LFRCO , false, false);

Disable the Low Frequency Crystal Oscillator, LFXO CMU_OscillatorEnable(cmuOsc_LFXO, false, false);

Route LF clock to LETIMER0 clock tree CMU_ClockSelectSet(cmuClock_LFA, cmuSelect_ULFRCO); // What clock tree does the LETIMER0 reside on?

CMU clock tree is established in this file
